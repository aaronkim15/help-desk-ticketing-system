# Help Desk Ticketing System

## Overview
A web-based help desk ticketing system that allows customers to report and track issues, which support agents can pick up and resolve transparently.

## Tech Stack
- frontend: JavaScript, HTML, CSS
- backend: Node.js, Postgres

## Project Structure

## Getting Started
### Prerequisites
- Git
- `npx serve`
### Installation
1. Click the green "Code" button on the repository
2. Copy the HTTPS web url
3. Run  `git clone [url]`
### How to Run
#### Frontend
1. In the main project directory, run `npx serve`
2. Open `localhost:3030` or as specified by `npx serve`
#### Backend
1. cd backend
2. npm install
3. npm run db:setup
4. npm start
